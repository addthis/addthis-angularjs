Tool Example 6 changing the default share title for all tools

Some services (such as Facebook) do not allow you to define the share title for a URL. Facebook will always use the Open Graph tags it finds on the page when it crawls it. You can use the Facebook Sharing Debugger to test your Open Graph tags.

You can test on the tool below, the sidebar on the right (on desktop) or the bar on the bottom (on mobile).

Any inline tool specifying a share URL using the share-title attribute will not be effected by this.

addthis_share.title === '{{currentTitle}}'
